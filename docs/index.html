01 CHOOSE THE RIGHT ISO

Download a clean ISO directly. Don't modify or debloat the ISO — waste of time.

VersionDescriptionBest For
IoT LTSC MAX PERFLightest of all. Extremely clean, zero bloatware.Enthusiasts, maximum performance
LTSC RECOMMENDEDSecurity updates only. Clean by default.Performance without hassle
Consumer Pro/HomeStandard. More stable but saturated with apps.Full Microsoft ecosystem
⚠

Windows 10: No longer receives feature updates. LTSC versions still get security patches and are an excellent lightweight choice.

⚡

Activation: MAS (Microsoft Activation Scripts) — free, open source.

📥 Download clean ISOs: massgrave.dev

// SECTION 02

02 PREPARE THE BOOTABLE USB

Use Rufus to write the ISO to your USB drive.

★

Pro tip: Use Ventoy for a multiboot USB — load multiple ISOs on one drive and pick at boot. Essential for any technician's toolkit.

For Windows 11, enable these Rufus bypass options:

rufus_options.cfg
  • Remove requirement for 4+ GB RAM, Secure Boot and TPM 2.0
  • Remove requirement for a Microsoft online account
  • Create a local account with a custom username
  • Disable data collection (skip privacy questions)
  • Disable automatic BitLocker device encryption

03 CLEAN GPU DRIVERS

Prepare slim drivers on the USB before installing Windows. Generic drivers cause conflicts.

🟢 NVIDIA — NVCleanstall

NVCleanstall — auto-detects best driver.

Components

  • Display Driver (required)
  • Legacy Control Panel
  • HD Audio via HDMI (only if needed)

Tweaks Tab

  • Disable Installer Telemetry & Advertising
  • Perform a Clean Installation
  • Disable Multiplane Overlay (MPO)
  • Unattended Express Installation
  • Add Hardware Support
  • Enable DLSS Indicator

→ Build Package → copy .exe to USB.

🔴 AMD — Radeon Slimmer

WHQL driver + Radeon Software Slimmer

Packages Tab

  • MS Visual C++ 2022 Redistributable 64bit
  • AMD HDMI Audio (skip if not needed)
  • All AMD ACP audio entries — REMOVE

Scheduled Tasks

  • All AMD auto-update tasks — DISABLE ALL

Display Driver Components

  • AMD Crash Defender (amdfendr.inf)
  • AMD-Windows Support Components
  • All others — REMOVE

→ Modify Installer → copy folder to USB.

// SECTION 04

04 INSTALL WINDOWS

1

Boot from USB and follow the normal Windows installation.

2

CRITICAL: When Windows is about to first start — disconnect Ethernet/WiFi. Don't let generic drivers install yet.

3

Windows 10 only: When prompted for a Microsoft account, press Shift + F10 and run:

OOBE\BYPASSNRO

PC reboots. Continue normally, create a local account.

4

Inside Windows, install prepared GPU drivers (quick install). Reboot.

5

Reconnect internet. Done.
