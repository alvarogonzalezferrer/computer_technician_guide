// SECTION 01

01 CHOOSE THE RIGHT ISO

Download a clean ISO directly. Don't modify or debloat the ISO - it's a waste of time. Pick your version below:

VersionDescriptionBest For
IoT LTSC MAX PERF Lightest version of all. Extremely clean, no bloatware whatsoever. Enthusiasts wanting maximum performance
LTSC RECOMMENDED Long-term support. Security updates only. Clean by default. Users wanting performance without hassle
Consumer Pro/Home Standard version. More stable but saturated with preinstalled apps. Users needing full Microsoft ecosystem
⚠

Windows 10 note: No longer receives feature updates. LTSC versions still get security patches. If you're comfortable managing your own security, Windows 10 LTSC is an excellent lightweight option.

⚡

Activation: Download and run MAS (Microsoft Activation Scripts) to activate Windows for free.

📥 Download clean ISOs from massgrave.dev

// SECTION 02

02 PREPARE THE BOOTABLE USB

Use Rufus to write the ISO to your USB drive. If installing Windows 11, enable these critical bypass options in Rufus before writing:

rufus_options.cfg
  • Remove requirement for 4+ GB RAM, Secure Boot and TPM 2.0
  • Remove requirement for a Microsoft online account
  • Create a local account with a custom username
  • Disable data collection (skip privacy questions)
  • Disable automatic BitLocker device encryption
✓

These options prevent BitLocker from silently encrypting your drive and Windows from forcing an online account - both cause serious headaches later.

03 CLEAN GPU DRIVERS

Before installing Windows, prepare a slim driver package on the USB. Generic drivers Windows installs on first boot create conflicts. Process differs by GPU brand:

🟢 NVIDIA - NVCleanstall

Download NVCleanstall - auto-detects best driver for your GPU.

Components to Install

  • Display Driver (required)
  • Legacy Control Panel
  • HD Audio via HDMI (only if you use audio via monitor/TV)

Tweaks Tab

  • Disable Installer Telemetry & Advertising
  • Perform a Clean Installation
  • Disable Multiplane Overlay (MPO)
  • Unattended Express Installation
  • Add Hardware Support
  • Enable DLSS Indicator

→ Click Build Package → copy the .exe to your USB.

🔴 AMD - Radeon Slimmer

Download WHQL driver + Radeon Software Slimmer

Packages Tab

  • Microsoft Visual C++ 2022 Redistributable 64bit
  • AMD HDMI Audio Driver (skip if no HDMI audio)
  • AMD High Definition Audio Controller
  • All AMD ACP audio entries

Scheduled Tasks Tab

  • AMDInstallLauncher - DISABLE
  • AMD COMPUTE - DISABLE
  • AMD Link Driver - DISABLE
  • All other auto-update tasks - DISABLE

Display Driver Components

  • AMD Crash Defender (amdfendr.inf)
  • AMD-Windows Support Components (amdwin)
  • All others - REMOVE

→ Click Modify Installer → copy output folder to USB.

// SECTION 04

04 INSTALL WINDOWS

1

Boot from the USB and follow the normal Windows installation process.

2

CRITICAL: When Windows is about to start for the first time - disconnect your Ethernet cable or disable WiFi. Don't let generic video drivers or updates get installed yet.

3

Windows 10 only: When prompted for a Microsoft account, press Shift + F10 and run:

OOBE\BYPASSNRO

The PC reboots. Continue normally and create a local account.

4

Inside Windows, install your prepared GPU drivers (quick install). Reboot.

5

Reconnect to the internet. Done.

Option B - ChrisTitus Script

Open PowerShell as Administrator and run:

irm "https://christitus.com/win" | iex

In the Tweaks tab, configure the following:

ESSENTIAL TWEAKS
  • Create Restore Point
  • Delete Temporary Files
  • Disable Activity History
  • Disable ConsumerFeatures
  • Disable Explorer Auto Folder Discovery
  • Disable Hibernation
  • Disable Location Tracking
  • Disable PowerShell 7 Telemetry
  • Disable Telemetry
  • Disable Win Platform Binary Table (WPBT)
  • Enable End Task With Right Click
  • Remove Widgets
  • Run Disk Cleanup
  • Set Services to Manual
ADVANCED TWEAKS
  • Disable Microsoft Copilot
  • Remove Microsoft Edge
  • Remove OneDrive
  • Remove Xbox & Gaming Components
  • Remove ALL MS Store Apps - NOT recommended
PREFERENCES
  • Dark Theme for Windows
  • Disable Multiplane Overlay
  • Modern Standby Fix
  • Show File Extensions
⚡

Don't skip this: In the Performance Plans section → click "Add and Activate Ultimate Performance Profile".

✓

If you removed Edge and have no browser: the first tab of the script lets you install Chrome, Firefox, Brave, etc.

Bonus - Overclock & Undervolt

TechniqueWhat It DoesResult
Overclock Increases CPU / GPU / RAM frequency above factory spec More raw performance
Undervolt Reduces voltage while maintaining performance Less heat, less power draw, more stability
ℹ

These are one-on-one because every component varies. No universal recipe - measure and adjust per your specific CPU/GPU.
